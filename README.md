# Projeto Integrador — Site para a ONG Gota de Amor

## 📌 Sobre o Projeto
Este projeto tem como objetivo desenvolver um site institucional para a ONG **Gota de Amor**, proporcionando uma presença digital mais profissional, acessível e informativa.

A proposta visa facilitar o contato entre a organização e a comunidade, permitindo maior divulgação de suas ações sociais, campanhas, eventos e formas de contribuição.

## 🎯 Objetivos
- Criar um site moderno, responsivo e acessível;
- Divulgar a missão, visão e projetos da ONG;
- Facilitar doações e captação de voluntários;
- Disponibilizar informações de contato e localização;
- Melhorar a comunicação com apoiadores e beneficiários.
